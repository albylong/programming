"Roll A Ball"

For this game I worked through the required tutorial step by step. First adding player field and writing the rules for it then doing the same for the player and it’s rules. Next was the camera which I added and linked to move correctly with the player object. I then worked through setting up the play area by adding walls and obstacles before moving on to collecting items to later be added to a score. To make this all work I had to then add in object collisions and detection for collectibles before finishing up by linking the collectables into the script which updates the score. Also needed to set the game to display “You Win” text when all collectables have been collected. Finally came adding the AI enemy to the game and exporting the game file to finish the tutorial.

2D Game

This game was heavily tutorial based. The purpose of this was to get used to Unity and navigating all of its functions. We worked through the Unity based tutorial and were taught how to use basic features when creating and editing 2D games. While we were given an almost finished base game we were left with the skills to edit this platformer game and make it somewhat our own. Overall this tutorial was valuable for getting us used to using the Unity engine for 2D game creation.
